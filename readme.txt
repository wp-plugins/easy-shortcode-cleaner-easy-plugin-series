Replace Unused Shortcodes with HTML Comment Tag automatically.

== Description ==

NO RISK. NO DB QUERY. KEEPING YOUR POSTED DATA.

You can use unused shortcodes again everytime.

ATTENTION: Some of your post replaces with HTML Comment Tag.
 e.g. `[this is example]` (start with [ and lowercase letters/numbers/_/-) 

[Easy Plugin Series for WordPress](https://sites.google.com/site/easypluginseries/)

== Installation ==

1. Upload `easy-shortcode-cleaner` to the `/wp-content/plugins/` directory and unzip
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enjoy!
